# New Project Checklist

## 1. Bootstrap
- [ ] Create repo from this template
- [ ] Run `.github/prompts/init-project.prompt.md`
- [ ] Fill `docs/specification.md`
- [ ] Confirm default workflow mode (Safe or Fast)

## 2. Project essentials
- [ ] Confirm deploy target
- [ ] Confirm rollback path
- [ ] Confirm required quality gates
- [ ] Remove irrelevant skills from `.github/skills/`

## 3. Team workflow
- [ ] PR template in use
- [ ] CI exists or is planned
- [ ] Branch policy agreed
- [ ] Main branch protection configured when relevant

## 4. Done when
- [ ] Project goal and scope are written
- [ ] One source of truth for workflow guidance is preserved
- [ ] First meaningful change can be made without restructuring the repo again
